# 变量

```js title:关键字 
{
var a = 1;
let b = 1;
}
var a = 2;
//let b = 3;   不可重复定义,会报错
//alter(b)     也会报错,b只在{}中存在
alter(a);  //可以获取

const p =3.14;   //常量
```
## var
弱类型
1. 作用域大,全局变量
2. 可以重复声明
## let
只在代码块有效
## const
常量
